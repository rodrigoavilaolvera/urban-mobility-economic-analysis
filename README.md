# Urban Mobility and Economic Analysis

## Project Overview
This project analyzes urban mobility datasets and economic indicators to identify patterns and relationships through exploratory data analysis and visualizations.

## Description
Performed data cleaning, exploratory data analysis, dataset merging, and aggregations using Python (pandas, NumPy, Matplotlib), creating visualizations to support analytical conclusions.

## Tools
- Python (pandas, NumPy, Matplotlib)

## Data Sources
Publicly available datasets related to urban mobility and economic indicators.
